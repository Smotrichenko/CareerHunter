# **CareerHunter**

## **Описание**
Проект для поиска и удобной работы с вакансиями с сайта hh.ru

## **Установка и запуск**
1. Клонируй репозиторий:

```
git clone git@github.com:Smotrichenko/CareerHunter.git
```
2. Установи зависимости через Poetry
```
poetry install
```
3. Запуск проекта
```
poetry run python main.py
```
## **Дополнительная информация по проекту**
### *Возможности:*

- Поиск вакансий по ключевому слову на hh.ru (через API).
- Сохранение вакансий в JSON-файл (накопительное хранилище без дублей).
- Просмотр, фильтрация и сортировка вакансий:
  - фильтр по ключевым словам в названии или описании;
  - фильтр по диапазону зарплаты;
  - сортировка по зарплате;
  - вывод топ-N вакансий.
- Удаление вакансии по ссылке.
- Простая архитектура (SOLID, наследование, инкапсуляция).
