# *CareerHunter*

## 📋 *Описание*
CLI-сервис для поиска и обработки вакансий с hh.ru через API.
Позволяет получать вакансии по ключевому слову, преобразовывать их в структурированные объекты, сохранять в JSON-хранилище без дублей и выполнять фильтрацию, сортировку и анализ данных.


## 🎯 *Возможности*
- поиск вакансий через API hh.ru
- преобразование ответа API в объекты вакансий
- сохранение вакансий в JSON без дублей
- фильтрация по ключевым словам
- фильтрация по диапазону зарплаты
- сортировка по зарплате
- вывод top-N вакансий
- удаление вакансии по ссылке

## 💡 *Какую задачу решает проект*
Проект автоматизирует первичный анализ вакансий:
- собирает подходящие предложения по ключевому запросу
- позволяет быстро фильтровать вакансии по зарплате и тексту
- упрощает локальное хранение и повторную работу с результатами поиска


## 🔔 *Пример использования*
1. Пользователь вводит ключевое слово: Python developer
2. Проект получает вакансии с hh.ru через API
3. Сохраняет результаты в JSON
4. Пользователь применяет фильтрацию по зарплате и ключевым словам
5. Получает top-N наиболее релевантных вакансий
```
{
        "title": "Python Developer (Django / FastAPI) — Middle/Senior",
        "url": "https://hh.ru/vacancy/129857480",
        "salary_from": 100000,
        "salary_to": 200000,
        "currency": "RUR",
        "description": "Коммерческий опыт разработки на Python от 2–3 лет. Уверенное знание Django и/или FastAPI. Опыт работы с Git и... Работа с микросервисной архитектурой. Проектирование, оптимизация и сопровождение баз данных (PostgreSQL, Redis). Интеграция с очередями сообщений и системами кеширования."
    }
```

## **Установка и запуск**
1. Клонируй репозиторий:

```
git clone git@github.com:Smotrichenko/CareerHunter.git
```
2. Установи зависимости через Poetry
```
poetry install
```
3. Запуск проекта
```
poetry run python main.py
```


## 🧪 *Тестирование*
Запуск тестов:
```
poetry run pytest
```
Покрытие:
```
poetry run pytest --cov=src
```
